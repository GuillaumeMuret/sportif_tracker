-> central_bleuart
Contient les fichiers sources permettant de programmer le module BLE Bluefruit Feather qui sera connecté par liaison série à
la BeagleBone.

-> peripheral_bleuart
Contient le fichier source permettant de programmer le module BLE Bluefruit Feather qui sera placé sur le joueur et connecté
au récepteur de fréquence cardiaque et à l'accéléromètre.

Requis : installation de l'IDE d'Arduino ==> https://www.arduino.cc/en/main/software
